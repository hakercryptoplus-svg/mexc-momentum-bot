## 📋 الميزات

- **مسح يومي** لـ 42 عملة في الساعة 00:05 UTC
- **شراء تلقائي** عند وجود Pump ≥ 5% (All-In) 🚀
- **TP +2%** و **SL -1%** — مراقبة كل 5 دقائق 🔄
- **🔄 Trailing Stop ذكي** — عند +1% الـ SL يتحرك لسعر الدخول (Breakeven) — حماية رأس المال 🛡️
- **تحكم كامل** عبر تيليجرام (أوامر عربية) 📱
- **إحصائيات** وتاريخ الصفقات 📊

---

## 🚀 طرق النشر

### ▶️ Render.com

1. إنشاء **New Worker** (وليس Web Service)
2. ربط المستودع من GitHub
3. ضبط المتغيرات البيئية (انظر الأسفل)
4. البوت يشتغل تلقائياً 🎉

### ▶️ Railway.app

1. **New Project** → **Deploy from GitHub repo**
2. إضافة المتغيرات في Dashboard
3. Railway يقرأ `Dockerfile` تلقائياً

### ▶️ يدوي

```bash
pip install -r requirements.txt
cp .env.example .env   # املأ المتغيرات
python bot.py
```

## 🔐 المتغيرات المطلوبة (Environment Variables)

| المتغير | الشرح | مثال |
|---------|-------|------|
| `TELEGRAM_TOKEN` | توكن البوت من @BotFather | `123456:ABC-DEF...` |
| `TELEGRAM_CHAT_ID` | آيدي حسابك في تيليجرام | `7281928709` |

**اختياري:**
| `BINGX_API_KEY` | مفتاح API من BingX (أو ضبطه عبر `/set_bingx`) |
| `BINGX_SECRET_KEY` | المفتاح السري |
| `MIN_PUMP` | أقل نسبة Pump للإشارة (افتراضي: 5.0) |
| `TAKE_PROFIT` | نسبة TP (افتراضي: 1.02 = +2%) |
| `STOP_LOSS` | نسبة SL (افتراضي: 0.99 = -1%) |
| `COINS` | قائمة عملات مخصصة (مفصولة بفواصل) |
| `SCAN_HOUR` | ساعة المسح اليومي (افتراضي: 0 UTC) |
| `SCAN_MINUTE` | دقيقة المسح (افتراضي: 5) |
| `STATE_DIR` | مسار حفظ حالة البوت (افتراضي: المجلد الحالي) |
